DESKRIPSI : 
Project ini adalah pembuatan automatic api testing dummyapi.io. 
Terdiri atas skenario:
1. Test get list data user
2. Test get list data user by id
3. Test create new user
4. Test create new user without First Name
5. Test create new user without email
6. Test update user
7. Test delete users
8. Test get list of tag

CARA RUN PROJECT: 
Project ini dapat dijalankan melalui CI/CD Github Action yang akan run otomatis setiap kali ada commit push ke repositori, pull request, merge branch, atau bisa dijalankan secara manual melalui menu github-workflow
